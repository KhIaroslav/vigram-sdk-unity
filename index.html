    viDoc SDK for Unity

    com.vigram.sdk-gh1 connects Unity mobile applications to viDoc devices over Bluetooth LE and exposes public APIs for GNSS/NMEA data, NTRIP corrections, laser measurements, device state, logging, and mobile motion data.

    Key Facts

    Parameter Value
    SDK version 1.0.0
    Unity package com.vigram.sdk-gh1
    Minimum Unity version 2021.3
    Target platforms iOS, Android
    Package path Packages/com.vigram.sdk-gh1
    Render pipeline Independent of Built-in, URP, and HDRP

    Audience

    This documentation is written for Unity developers integrating a viDoc device into a mobile application, QA engineers validating Bluetooth/GNSS/NTRIP behavior, and SDK maintainers who need a clear map of the public API surface.
