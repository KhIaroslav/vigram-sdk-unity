    LaserConfiguration

    Creates a configuration message to be sent to the laser.

    • Parameters:
      • shotMode: The shot mode to be used for recording individual laser distances. Specifying a faster shot mode will lead to more individual distance recordings in the given duration.
      • position: The position of the laser to be used for laser recordings. In case there are multiple lasers, you can specify, which laser to use.
      • duration: The overall duration of the recording session.
